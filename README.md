# Project "Chat (Slack)":

[![Actions Status](https://github.com/sobolevaea/frontend-project-12/actions/workflows/hexlet-check.yml/badge.svg)](https://github.com/sobolevaea/frontend-project-12/actions)
[![Maintainability Rating](https://sonarcloud.io/api/project_badges/measure?project=sobolevaea_frontend-project-12&metric=sqale_rating)](https://sonarcloud.io/summary/new_code?id=sobolevaea_frontend-project-12)

A simplified Slack-like chat application built as a diploma project showcasing modern frontend development practices and technologies used by React developers.

## Demonstration

**Live version**: https://frontend-project-12-rojw.onrender.com

This project demonstrates the complete spectrum of standard tasks faced by frontend developers, including:

- WebSocket integration for live messaging
- REST API interaction
- Functional components with hooks
- Redux via Redux Toolkit for scalable application state
- Efficient navigation and routing
- Secure user access control
- Vite for bundling and production deployment

## Tech Stack

### Core Technologies

- **React** with Hooks, Context API, and performance optimization techniques
- **Redux Toolkit** - Standard state management library for React applications
- **React Router** - Client-side routing for single-page applications
- **Vite** - Next-generation frontend build tool for fast development and bundling

### Additional Libraries

- **Socket.IO Client** - Real-time, bidirectional communication for live data updates
- **Axios** - Promise-based HTTP client for making API requests
- **Formik + Yup** - Form handling library for complex form validation and management
- **React Bootstrap** - UI component library for Bootstrap-based interfaces
- **react-i18next** - Internationalization framework for multilingual applications
- **leo-profanity** - Library for filtering and sanitizing profane words
- **react-toastify** - Library for displaying toast notifications
- **Rollbar** - Error tracking and monitoring service for production applications

## Getting Started

Follow these steps to set up the project locally:

1. Clone the repository
```bash
git clone git@github.com:sobolevaea/frontend-project-12.git
cd frontend-project-12
```

2. Install dependencies
```bash
make install
```

3. Start the development server
```bash
make start
make run-frontend
```

The application will open in your browser at http://localhost:5002/
